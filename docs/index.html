AI Didattica – Documentazione

Libreria Java per lo studio didattico dell’Intelligenza Artificiale (regressione lineare e reti neurali).

InputValue

Descrizione: Rappresenta l’input di un modello. Può essere scalare (1 valore) o vettoriale (più valori).

Costruttori

Firma Descrizione
InputValue(double x) Crea un input scalare.
InputValue(double... x) Crea un input vettoriale (ND).

Metodi pubblici

Firma Descrizione
double get() Restituisce il primo valore (caso 1D).
double[] getVector() Restituisce l’input come vettore.

Model

Descrizione: Classe astratta base per tutti i modelli di apprendimento.

Metodi pubblici

Firma Descrizione
double predict(InputValue x) Calcola la predizione per un singolo input.
double[] predictAll(InputValue[] X) Calcola le predizioni per più input.

Trainable

Descrizione: Interfaccia per i modelli che possono essere addestrati.

Metodi

Firma Descrizione
void train(InputValue[] X, double[] Y, double lr, int epochs) Addestra il modello su un dataset.
double loss(InputValue[] X, double[] Y) Calcola l’errore MSE sul dataset.

LinearRegression1D

Descrizione: Modello di regressione lineare del tipo y = m·x + q.

Caratteristiche

  • Output sempre scalare
  • Addestramento con discesa del gradiente
  • Usa MSE come funzione di errore

NeuralLayer

Descrizione: Rappresenta un vero layer neurale: pesi + bias + funzione di attivazione.

Costruttore

Firma Descrizione
NeuralLayer(int in, int out, Activation a) Crea un layer con in ingressi e out neuroni.

NeuralNetwork

Descrizione: Rete neurale feed-forward composta da più NeuralLayer.

Esempio

NeuralNetwork nn = new NeuralNetwork();
nn.add(new NeuralLayer(2, 3, Activation.SIGMOID));
nn.add(new NeuralLayer(3, 1, Activation.SIGMOID));
